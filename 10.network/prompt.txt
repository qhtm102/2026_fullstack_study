네이버 open api 호출해서 도서 목록을 조회하는 리액트 페이지 개발중이야.
한 건의 데이터는 아래와 같은 json 구조로 이루어져 있어
조회결과를 목록으로 표시하는 컴포넌트와 각 항목을 표시하는 컴포넌트 화면을 만들어줘.
기능은 구현하지 말고 화면만 구현해줘.
컴포넌트는 src폴더 내부에 component 폴더를 만들고, component 폴더 안에 컴포넌트를 저장해줘.
각 컴포넌트는 BookSearchList.jsx와 BookSearchItem.jsx로 만들어줘.
