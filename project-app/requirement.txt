react 기반 앱을 개발할거야.
상장회사를 검색해서 기업 정보를 표시하고 해당 기업의 주가 정보를 
아래의 요구사항을 반영해서 앱을 구현해줘 

1. 기업 정보 페이지 구현 (pages/Company.jsx)

   - 상장회사 검색 컴포넌트 구현 (components/SearchCompany.jsx)

      사용자가 회사 이름을 입력하고 검색 버튼을 클릭하면 검색 결과를 표 형태로 표시
      API 사용 : http://192.168.0.19:8000/api/v1/stocks/search?name={company_name}

   - 상장회사 정보 표시 컴포넌트 구현 (components/CompanyInfo.jsx)

      상장회사 검색 컴포넌트의 검색 결과 목록 중에서 회사를 선택하면 해당 회사 정보 표시
      API 사용 : http://192.168.0.19:8000/api/v1/stocks/{code}
      주식코드를 파라미터로 주가정보 페이지로 이동하는 링크 추가

   - 페이지 왼쪽에 상상회사 검색 컴포넌트, 오른쪽에 상장회사 정보 표시 컴포넌트 배치


2. 주가정보 페이지 구현 (pages/Stock.jsx)

   - 주가 검색 컴포넌트 구현 (components/QuotesSearch.jsx)

     주식코드, 시작일자, 종료일자를 입력하고 클릭하면 해당 기간의 주가정보 조회
     날짜가 하나라도 지정되지 않은 경우 최근 1년간의 주가정보 조회
     API 사용 : http://192.168.0.19:8000/api/v1/stocks/{code}/prices
     조회가 완료되면 주가 정보 목록 표시 컴포넌트와 주가 정보 그래프 표시 컴포넌트 업데이트

   - 주가 정보 목록 표시 컴포넌트 구현 (components/QuotesList.jsx)

     주가 검색 컴포넌트에서 조회된 주가 정보를 그래프 형태로 표시      

   - 주가 정보 그래프 표시 컴포넌트 구현 (components/QuotesChart.jsx)

     주가 검색 컴포넌트에서 조회된 주가 정보 중 가격은 선그래프로 거래량은 막대그래프로 표시
     rechart library 사용하며 이미 설치되어 있으므로 import만 하면 됨

   - 페이지 상단에 주가 검색 컴포넌트, 왼쪽에 주가 정보 목록 표시 컴포넌트, 오른쪽에 주가 정보 그래프 표시 컴포넌트 배치
       

3. 라우팅 구현

   react-router-dom 기반 라우팅 사용
   라이브러리는 이미 설치되어 있으므로 import만 하면 됨
